GitHub started in 2008 and quickly became one of the most popular places for developers to share and work on code together.

Beyond just hosting code, it offers tools like Actions for automation, and Codespaces for instant development environments.

People use GitHub to work together, learn from each other, and build software in the open.

Open Source Projects

Whether you're browsing cool side projects or contributing to major frameworks, GitHub makes it easy to get involved.

Built-In Security

It checks your code for known vulnerabilities and helps you fix them with helpful suggestions.

Developer Network

GitHub is home to a massive global community of developers who share ideas and solve problems together.
