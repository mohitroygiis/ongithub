New to Git? Here’s a quick guide to some of the most useful commands you'll use almost daily.

git init

Start a new Git repository in your current folder.

git clone [url]

Make a copy of a repository that's hosted remotely, like on GitHub.

git add [file]

Tell Git which files you want to track or update in your next commit.

git commit -m "message"

Save your changes with a short, meaningful message describing what you did.

git push

Send your commits to the remote repository (like GitHub) so others can see them.

git pull

Get the latest changes from the remote repository and merge them into your local copy.

git status

See which files have changed and what Git is tracking.

git branch

List all your branches. Branches are like alternate timelines for your project!

Want to learn even more? Git’s official documentation is full of tips: git-scm.com/doc
